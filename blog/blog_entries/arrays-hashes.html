Hash vs Arrays

In Ruby, while both Arrays and Hashes are ordered collections of elements, these two objects have key differences:

An Array is ordered through a consecutive numerical index. >> Star_Wars_Sequels = Array.new(["Episode IV","Episode V", "Episode VI"]) => [Episode IV, Episode V, Episode VI] As can be seen, the Array is used to sequentially order the objects of the Array.

A Hash is ordered through being stored in pairs, each consisting of a key and a value. For example: >> ratings = Hash.new => ratings["Revenant"] = 10 In this example, we're not ordering the Hash object sequentially, we are merely assigning a rating value to the movie Revenant.
