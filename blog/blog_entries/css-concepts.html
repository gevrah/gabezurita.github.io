Display: Inline vs. Inline-Block

The display property allows one to turn an inline element into a block element and vice-versa.

The following image is a visual representation of the difference between inline elements and inline block elements (source):

To expound on the above: both Inline and inline-block elements allow other elements to sit to their left and to their right, although they have a few key differences. Inline elements respect left and right margins as well as padding, but they do not respect top and bottom. They also cannot have their height or width set. On the other hand, as can be seen above, inline-block elements do respect top and bottom margins/padding, and they respect height and width.

An example of an inline element would be a span, which is a hook that occupies only the left and right space bounded by the tag. It does not respect top and bottom. By giving an element the CSS property of { display: inline }, one can make said element behave like a span.
