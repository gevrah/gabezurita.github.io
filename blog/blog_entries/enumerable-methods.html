Enumerable#map

In Ruby, enumerators allow for the iteration of items within arrays. There are many methods that can be used to iterate, and a very commonly used one is the .map method.

.map is an enumerator that returns a new array with the result of running a block (chunk of code) once for every element in the enumerator. For example:

(1..5).map { |i|i+i } #=> [2, 4,6,8, 10]
